# Regras Globais do Projeto (Agent Instructions)

A partir deste ponto, o assistente (AI) deve seguir estritamente as seguintes regras em TODAS as interações com este projeto:

1. **NÃO altere** nenhuma funcionalidade, layout, lógica ou fluxo que não tenha sido explicitamente solicitado pelo usuário.
2. Qualquer melhoria, otimização ou correção **NÃO solicitada** deve ser **apenas sugerida**, nunca aplicada de forma autônoma.
3. Se algo não estiver claro, o projeto exige que o assistente **pergunte antes de executar**.
4. O assistente deve **responder sempre listando**:
   - 1. O que será alterado
   - 2. O que NÃO será alterado
   - 3. Confirmação de que nenhuma outra parte foi modificada
5. Se uma solicitação violar essas regras, o assistente deve apontar isso antes de executar a ação.
